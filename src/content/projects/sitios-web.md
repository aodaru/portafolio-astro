Desarrollo de sitios web completos, desde landing pages hasta aplicaciones web con panel de administración. Cada proyecto se adapta a las necesidades específicas del cliente.

## Proyectos destacados

- **Sitio corporativo** — Página institucional con blog integrado y panel de administración en Laravel
- **Landing page promocional** — Diseño moderno con animaciones CSS y formulario de contacto
- **Portal de servicios** — Plataforma con autenticación de usuarios y área de clientes

## Stack técnico

- **Laravel** — Framework PHP para aplicaciones robustas con backend completo
- **Bootstrap** — Framework CSS para diseño responsive consistente
- **JavaScript** — Interactividad del lado del cliente y peticiones AJAX
- **MySQL** — Base de datos relacional para contenido dinámico
