Sistema de intranet desarrollado para la gestión interna de la empresa, automatizando procesos clave como facturación, control de inventario, directorio de empleados y reportes de ventas.

## Funcionalidades principales

- **Facturación por código** — Sistema de facturación ágil con búsqueda por código de producto
- **Directorio de empleados** — Gestión de datos del personal con perfiles y permisos
- **Reportes de ventas** — Dashboard con métricas en tiempo real y exportación de datos
- **Creación de materiales** — Módulo para alta y edición de productos con imágenes y precios
- **Impresión de ofertas** — Generación de plantillas personalizadas para cotizaciones

## Stack técnico

- **PHP** — Lógica del backend y conexión a base de datos
- **MySQL** — Almacenamiento relacional de datos
- **JavaScript** — Interactividad en el frontend y validación de formularios
- **HTML5 + CSS3** — Interfaz de usuario responsive
