El objetivo de este proyecto es llevar un historial de los trabajos en los cuales he realizado algún tipo de desarrollo, como también compartir las experiencias vividas en estos proyectos y algunas otras experiencias y dificultades que se me hayan presentado en este recorrido de aprendizaje en el desarrollo web y otras herramientas.

## Stacks

- **Laravel** — Framework de desarrollo para PHP.
- **Angular** — Moderna plataforma de desarrollo web.
- **Bootstrap** — Facilita la creación de sitios web responsivos.
- **MomentJS** — Interpreta, valida, manipula y muestra fechas con formatos personalizados en Javascript.
- **Front-Matter** — Agrega capacidad de metadatos a los documentos Markdown (título, descripción, etc.).
- **ngx-markdown** — Librería de Angular para visualizar Markdown.
