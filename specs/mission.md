# Mission

## Propósito
Migrar el tema WordPress "gbaod" (GruvBox-based) a un sitio web Astro moderno, estático y rápido, usando archivos Markdown para el contenido del blog y portafolio.

## Visión
Un sitio personal híbrido blog + CV que muestre la trayectoria profesional, proyectos técnicos y artículos de Adal Michael García, ingeniero en sistemas de Panamá.

## Audiencia objetivo
- **Reclutadores** — que busquen ver perfil técnico, experiencia, proyectos.
- **Desarrolladores** — que encuentren artículos técnicos, código, recursos.
- **Audiencia general** — que conozcan el portafolio, bio, pasatiempos y formas de contacto.

## Valores de diseño
- **Rediseño completo** — no mantener el tema WP tal cual; se busca un look fresco y moderno.
- **Rendimiento** — sitio 100% estático, fast loads, sin dependencias pesadas.
- **Accesibilidad** — WCAG básico, navegación por teclado, contraste suficiente.
- **Bilingüe** — Español e inglés, con ruteo por idioma.

## Contenido a migrar
- Bio personal con foto
- Cronología profesional
- Trabajos/Proyectos destacados
- Blog con artículos
- Tags para los posts
- Dark/Light mode
- Contacto (redes sociales)
